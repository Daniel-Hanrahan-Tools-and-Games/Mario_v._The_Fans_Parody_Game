# Mario_v._The_Fans

A parody text-based RPG where Mario from the Super Mario Bros series fights his Fans.

I have no plans on releasing this on fantasy computers, Game Creation systems, Scratch or having mod support in this game but anyone can make those versions or add that stuff in the game if they want to.

The game is a text-based game, where the game says 1 in text fields replace that with Mario and where the game says 2 in text fields replace that with specific fans or types of fans. 

To execute executable on linux, you open up terminal then you drag the executable into terminal then press enter.

<h3>How to compile the video game adaptation with g++</h3>
<ol>
  <li>Put this in the terminal first then press enter: g++ /"Path to the folder"/Mario_v._The_Fans_Parody_Game/Mario_v._The_Fans_Parody_Game.cpp -o /"Path to the folder"/Mario_v._The_Fans_Parody_Game/Mario_v._The_Fans_Parody_Game</li>
  <li>The executable file will be in your Game directory</li>
  <li>To open the executable you just drag the executable into your terminal.</li>
</ol>
